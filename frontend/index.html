Décrivez votre situation de travail
Pour générer une analyse de risques précise et exploitable, renseignez simplement les éléments ci-dessous. Vous pouvez répondre de manière simple, même en vrac : l'outil structure automatiquement vos informations.

Déposez vos informations (même en vrac)

Champs / aide à afficher
1. Activité / métier : secteur, type d'activité, produit ou service
2. Poste concerné : intitulé du poste + nombre de personnes exposées
3. Environnement de travail : atelier, chantier, bureau, extérieur, coactivité, etc.
4. Tâches réalisées : 3 à 10 tâches réelles, fréquence, durée approximative
5. Équipements et produits utilisés : machines, outils, véhicules, produits chimiques, etc.
6. Contraintes de travail : horaires, pression, chaleur, bruit, travail isolé, sous-traitance…
7. Incidents / situations : accidents, presqu'accidents, pannes, situations dangereuses (si applicable)
Vous pouvez remplir seulement les éléments que vous connaissez. Plus les informations sont précises, plus l'analyse sera pertinente.
